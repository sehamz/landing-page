# Landing Page Project

Landing page project is the second project in Udacity Front-End Development Nanodegree.

The aim of this project is to:
- Add a dynamic navigation bar contains links of the sections. And according to the current section in the viewport, the proper link will be highlighted. 
- Add "active class"(CSS style) to the current section in the viewport to make it stand out.
- Apply smooth scroll whenever the user click on the navigation bar to jump to any section.

The project contains the modified HTML&CSS code and the new js file.
